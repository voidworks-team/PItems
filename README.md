# PItems

**PItems** is a **Partner Items** plugin for **HCF** servers running on PocketMine-MP.

It allows you to create, manage, and use special items with unique effects, designed for combat and team support mechanics.

## Features

- Custom partner item system
- Supports combat and interaction-based usage
- Commands to give items to players
- Modular structure for easy expansion
- Built for HCF gameplay

## Requirements

- PocketMine-MP 5.x
- PHP 8.2 or higher
- Composer dependencies installed

## Installation

1. Download the plugin.
2. Place the `.phar` file into your server's `plugins/` folder.
3. Start or restart the server.
4. Adjust the configuration to fit your needs.

## Usage

The plugin includes commands to manage and distribute partner items to players.
